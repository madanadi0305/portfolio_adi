## 🚀 Deployment

This portfolio is a single static HTML file hosted on **Vercel**.

### Deploy your own copy

1. Fork or clone this repo
2. Rename `portfolio.html` → `index.html`
3. Go to [vercel.com](https://vercel.com) and sign in with GitHub
4. Click **Add New → Project** → Import this repo
5. Leave all settings as default → click **Deploy**

Every `git push` to the main branch will automatically redeploy the live site.

---

## 🛠️ Local Development

No build tools needed. Just open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/madanadi0305/portfolio.git

# Open in browser
open index.html
```

---

## 📝 Adding a New Project

1. Open `index.html`
2. Find the relevant tab section (Data Analyst / Data Engineering / AI Projects)
3. Copy an existing `<a class="card ...">` block and update the title, description, tags, and `href` link
4. Save and push to GitHub — Vercel auto-deploys within seconds
